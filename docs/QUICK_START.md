# 快速开始指南

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行游戏

### 图形模式
```bash
python main.py
```

### Headless 模式 (无GUI测试)
```bash
python main.py --headless
```

## 基本操作

### 摄像机控制
- **W/A/S/D** 或 **方向键**: 移动摄像机
- **鼠标滚轮**: 缩放
- **Space**: 暂停/继续
- **1/2/3**: 设置时间速度 (1x/2x/5x)

### 区域设置
1. 按 **S** 键进入仓库放置模式
2. 在地图上**右键点击**放置仓库区域
3. 按 **X** 键退出放置模式

其他区域类型:
- **F**: 农场区域
- **R**: 住宅区域

### 观察村民
- **左键点击**: 选择实体/格子
- **右下角面板**: 查看选中实体的详细信息
- **右上角面板**: 查看游戏状态和区域模式

## 测试场景

运行 headless 模式会自动执行测试:
1. 创建村民和树木
2. 设置仓库区域
3. 创建砍树任务
4. 村民自动砍树并搬运到仓库

测试成功会显示: `TEST SUCCESS: Log hauled to stockpile!`

## 配置文件

编辑 `config/balance.json` 可以调整游戏数值:
- 村民移动速度
- 砍树速度
- 技能初始值
- 等等

配置文件支持热重载，修改后自动生效。
